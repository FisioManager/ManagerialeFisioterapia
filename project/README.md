- `npx webpack`, build
- `npm start`, debug react
- `npm run server`, run server

## <u>MongoDB</u>

- `mongod --dbpath .`, runnare all'interno di /db per far partire il server
- `mongo`, entrare nella shell di mongodb
- `use databaseName`, selezionare database
